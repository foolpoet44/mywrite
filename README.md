# mywrite

## 시스템 가이드

이 리포지토리는 'Capture', 'Connect', 'Branch', 'Merge'의 4단계 흐름을 따르는 지식 관리 및 글쓰기 시스템입니다.

### 폴더 구조

- **inbox/**: [Capture] 구글 킵 등에서 수집된 날것의 메모들.
- **garden/**: [Connect] 메모들이 연결되어 발전된 아이디어들.
- **drafts/**: [Branch] 실제 집필 중인 초안 (에세이, 비평 등).
- **archive/**: [Merge] 완료된 글 (월별 또는 카테고리별).
- **assets/**: 이미지, 도표, 참고 PDF, 녹음 파일 등의 리소스.
- **.github/**: 깃허브 설정 파일.
